## 🚩 주요기능
- 각 주 후보 지지율 평균치 확인 가능
- 최근 10주 지지율 추이 그래프 확인 가능

## 💯 지지율 산정 방법
```
각 조사의 지지율 평균을 산정, 특히 표본수가 많은 여론조사에 더 많은 가중치를 두어 계산
```
